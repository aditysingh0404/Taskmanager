TASK MANAGER APPLICATION(JAVA)

OVERVIEW

This is a simple **Task Manager application** developed using Java.
It helps users manage daily tasks by allowing them to add, view, and mark tasks as completed.the project is designed for beginners to understand basic programming concepts and data structures in java.

This project is created as part of the **Bring Your Own Project (BYOP)** for the Programming in Java course.


PROBLEM STATEMENT

Students often forget assignments, deadlines, and important activities.
Managing tasks manually can be inefficient and unorganized.


 SOLUTION

This application provides a simple command-line interface to:

* Add tasks with deadlines
* View all tasks
* Mark tasks as completed


  FEATURES

*   Add new tasks
*  View all tasks
*  Mark tasks as completed
*  Exit program safely


TECHNOLOGIES USED

* Java (Core Java)
* Object-Oriented Programming (OOP)
* ArrayList
* Scanner (for user input)

HOW TO RUN

Step 1: Compile the program

javac *.java


 Step 2: Run the program


java Main


 SAMPLE OUTPUT


Task Manager 
1. Add Task
2. View Tasks
3. Mark Task Complete
4. Exit
